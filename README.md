VisionVoice 🎙️
AI-Powered Image Captioning with Live Camera & Audio Output
PBL Project by Deepesh

What is VisionVoice?
VisionVoice is an AI system that:

Clicks — Opens your webcam and captures a live photo on SPACEBAR
Understands — Runs the photo through a Deep Learning model (InceptionV3 + LSTM) to generate a natural language caption
Speaks — Reads the caption out loud using text-to-speech

How the Model Works
[Webcam] → Capture Photo → InceptionV3 (CNN) → Feature Vector (2048,)
                                                          ↓
                                                 Encoder Dense (512)
                                                          ↓
                                  [startseq] → Embedding → LSTM → Next Word
                                                          ↓
                                              Repeat until 'endseq'
                                                          ↓
                                               Speak Caption Aloud

Step-by-Step Setup & Run
Step 1 — Extract Image Features
bashpython extract_features.py
Reads all images from data/Images/ and saves CNN features to features.pkl

Step 2 — Preprocess Captions
bashpython preprocess.py
Reads data/captions.txt, tokenizes all captions, saves tokenizer.pkl

Step 3 — Train the Model
bashpython train.py

Trains a 512-neuron LSTM model for 30 epochs
Saves best model as vision_voice_model.keras
Automatically resumes if a saved model already exists
Target loss: below 3.0


Step 4 — Run the Demo
bashpython main.py
What happens:
Step 1 → Webcam opens with live preview
Step 2 → Press SPACEBAR to click the photo
Step 3 → Photo saved to data/Images/captured.jpg
Step 4 → AI analyzes the photo and generates a caption
Step 5 → Captured image shown with caption as title
Step 6 → Caption spoken out loud
